## Overview
Dictor bot is the Telegram bot that reads received messages for you. It synthesies text received in the message into speech and replies with a voice message.

![image](https://user-images.githubusercontent.com/2770755/162708779-8c35f742-651a-448b-9e99-015e1e5b47ce.png)

## Functional capabilities
- **Text-to-speech**. Converts text messages into voice messages
- **Multilingual**. Supports English and Russian languages
- **Configurable**. Supports different voices and quality
